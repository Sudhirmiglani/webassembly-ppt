Example code of Js using dynamic variables eg. var a = 10; var b = a; console.log(b); Problem : We don't know the type of b.
One Solution : eg. var a = 10; var b = a | 0; console.log(b); Explanation : This will always make sure that b will always be a 32 bit int
One Solution : eg. var a = 10; var b = a | 0; console.log(b); Explanation : This will always make sure that b will always be a 32 bit int This is ASM.JS
Now What is WebAssembly then?
It is an extension to ASM.js. WebAssembly defines an Abstract Syntax Tree (AST) that gets stored in a binary format. Binary is great because it means we can create smaller app bundles.
WebAssembly fills in the gaps that would be awkward to fill with JavaScript.
Where going to use WebAssembly
Games, VR, Augmented Reality
